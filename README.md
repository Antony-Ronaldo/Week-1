# AstraWorks – Web Development Portfolio

A simple and modern portfolio webpage built using semantic HTML and clean CSS.  
This project demonstrates the fundamentals of responsive web design and visual layout.

---

## Project Description

AstraWorks is a beginner-friendly web portfolio created to practice core web development concepts.  
It focuses on structure, styling, and responsiveness without using any frameworks or libraries.

The design follows a dark theme with a split hero layout, skill cards, and a minimal contact section.

---

## Features

- Semantic HTML structure
- Modern CSS styling with variables
- Responsive layout for different screen sizes
- Clean navigation and section-based layout
- Simple JavaScript for dynamic year display

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (basic)
